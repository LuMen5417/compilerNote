只有具备这些基本元素才能算为一个完备的脚本语言,暂定这些基本元素，后期在有需要再进行进一
步的修改：
\begin{itemize}
	\item 支持四则运算
	\item 支持字符串
	\item 支持变量
	\item 一些其它特性,如不支持‘;’
	\item 支持基本逻辑关键字
\end{itemize}
